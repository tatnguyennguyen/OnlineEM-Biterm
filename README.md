# OnlineEM-Biterm
Online EM learning for Biterm Topic Model
# Build
## Prerequisite
You need *cmake* and *armadillo* library to build this project

With Debian/Ubuntu:
```
[sudo] apt-get install cmake libarmadillo-dev
```
## Build the code
```
mkdir build
cd build
cmake <path_to_project_folder>
make
```
